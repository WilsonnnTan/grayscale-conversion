# Grayscale Image Algorithm

This project implements a grayscale image conversion algorithm.

## Algorithm Source

Implementation based on the grayscale algorithm described in:
- **Article**: [Grayscale Image Algorithm](https://tannerhelland.com/2011/10/01/grayscale-image-algorithm-vb6.html)  
- **Author**: Tanner Helland

## Overview

The project implements efficient methods for converting color images to grayscale format using established image processing techniques.

## How To Run Project

### Install uv (if not already installed)

For Windows, run in PowerShell:
```powershell
powershell -c "irm https://astral.sh/uv/install.ps1 | more"
```

### Install dependencies

```bash
uv sync
```

### Run project file

```bash
uv run main.py
```
